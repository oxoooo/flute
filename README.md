# Flute

一个简单的 ruby 内容爬虫框架，惯例优先，提供脚手架功能。

注意：
- 框架性能一般般
- 目前没提供 ip 池服务
- 本项目从我馒头好不好的爬虫框架提取而来，所以可能抽象没有那么高级别

## Installation

    $ gem install flute

之后可以使用 `flute new project_name` 来创建一个项目。

## Usage

项目包含了一个从爱范儿爬虫例子，可以进行参考，或许会有更多的实际例子发布
