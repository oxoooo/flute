# Flute

一个简单的 ruby 内容爬虫框架，惯例优先，提供脚手架功能。

注意：
- 框架性能一般般
- 目前没提供 ip 池服务
- 本项目从我馒头好不好的爬虫框架提取而来，所以可能抽象没有那么高级别

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'flute'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install flute
